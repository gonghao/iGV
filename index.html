Features: 

 Based on all the features of GraceMode of autoddvpn.

  • CDN works magnificently.
  • P2P works magnificently .   
  • Speed up Google services. (Looks like there will be a huge change soon.)
  • Speed up App Store downloads.
  • Speed up Amazon EC2/Tumblr/Cloudapp. (Need more IP address ranges of Tumblr and Cloudapp.)
  • No DNS pollution.
  • Easy to speed up foreign sites and block sites.Easy to DIY! :D

Install

You can also check the blog post in Chinese which may be outdated.

  • Install Xcode
  • Install Homebrew
  • Install Git (if you like)
    • $ brew install git
  • Update Homebrew
    • $ brew update
  • Install dnsmasq
    • $ brew install dnsmasq

You can refer to these two Screenshots:

    • Install Homebrew and update.
    • Install dnsmasq.
  • Download files
    • $ git clone git://github.com/xream/iGV or download the zip/tar.
  • Edit resolv.conf. Use your ISP's DNS servers or just use the 114DNS.
  • $ sudo chmod a+x ip*
  • $ sudo cp ip* /etc/ppp
  • $ sudo cp dnsmasq.conf /usr/local/etc/dnsmasq.conf
  • $ sudo cp resolv.conf /usr/local/etc/resolv.conf
  • Reboot.
  • Change the DNS settings of your VPN to 127.0.0.1.
  • Dial up the VPN and enjoy! :D

Tips

  • Is dnsmasq running?
    • $ ps -A | grep dnsmasq
  • To block sites
    • Edit dnsmasq.conf. Add lines like: address=/baidu.com/0.0.0.0
